# StudyBuddy App

## Project Goals
The goal of this project is to design and develop a mobile application using React Native that helps students effectively manage their academic responsibilities. The app will allow users to organize assignments, track deadlines, and stay productive through a simple and intuitive interface.

## Features
- User-friendly mobile interface
- Create, edit, and delete assignments/tasks
- Set and manage due dates
- Mark tasks as completed
- View tasks in a centralized dashboard
- Optional notifications for upcoming deadlines

## Overall Architecture
The application will be built using React Native.

- Frontend: React Native (screens, components, navigation)
- State Management: Handles task data and user input within the app
- Backend (optional): Firebase or Node.js for storing data
- Database (optional): Cloud storage for saving user tasks

The app will primarily function as a client-side mobile application, with optional backend integration.

## Team Collaboration
We will use GitHub for version control and collaboration.

- Each team member will work on assigned features
- Code will be pushed regularly to the repository
- Progress will be tracked through commits
- Communication will happen through group chat and regular check-ins

## Work Distribution

### Member 1 – Frontend / UI
- Build app screens (Home, Task List, Add Task)
- Design layout and styling
- Implement navigation between screens

### Member 2 – Functionality / Logic
- Implement add/edit/delete task features
- Manage app state and data flow
- Handle task completion and validation

### Member 3 – Testing / Documentation
- Test features and find bugs
- Fix minor issues
- Maintain README and documentation
- Help integrate all parts of the app
- Prepare demo and presentation
